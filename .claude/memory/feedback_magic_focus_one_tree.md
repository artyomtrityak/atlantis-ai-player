Focus magic study on a single tree and go as deep as possible. Do not branch into a secondary magic school unless the primary is fully exhausted.

**Why:** Each mage can only CAST once per turn. If Seraphel has SUSK (summon skeletons) and also studies DEMO (demon lore), she can still only cast one spell per turn — either SUSK *or* a demon spell, never both. Branching means studying skills you can never simultaneously use, wasting study months.

**How to apply:** For Seraphel (and any summoner mage), the priority is:
1. Study deeper into NECR (necromancy) → unlocks SUSK 2, SUSK 3, etc. (more skeletons per cast)
2. Only consider a secondary school (e.g., FEAR, FSHI) if the primary tree is fully gated by prerequisites that take multiple turns to unlock
3. When a mage has SUSK and NECR, the next study should be NECR (to raise level → raises SUSK cap), not a new school
